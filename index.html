Download Edit Publish

Build your website.

3 steps. No coding experience needed.

1 Download

Download ZIP

Unzip it. Open the folder.

2 Edit

Open starter.html in any text editor (VS Code or Notepad).

Replace the placeholder text with your info:

Your Nameyour name
your.email@example.comyour email
yourusernameyour GitHub username

Double-click starter.html to preview in browser.

3 Publish

A. Upload to GitHub

  1. Create account at github.com/signup
  2. Go to github.com/new and create repository
  3. Rename starter.html to index.html
  4. Click "uploading an existing file" then drag your files in and click Commit

B. Deploy on Cloudflare

  1. Create account at cloudflare.com
  2. Click Workers and Pages in sidebar
  3. Click Create application
  4. Click Continue with GitHub
  5. Select your repository
  6. Enter build settings:
    • Build command: npm run build
    • Build output: dist
  7. Click Deploy
  8. Done. Your site is live.

Troubleshooting

Changes not showing? Save the file (Ctrl+S), refresh browser (F5).

Site looks broken? Make sure CSS file is named styles.css.

Build failed? Check build command is npm run build and output is dist.

Questions? Ask the workshop facilitator.
