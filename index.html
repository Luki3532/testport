3

Publish to the Internet

3A. Create a GitHub Account

  1. Go to github.com/signup

    Create a free account (you'll use this username for your site)

3B. Upload Your Files to GitHub

  1. Go to github.com/new

    Create a new repository (name it anything, like my-website)

  2. Click "uploading an existing file"

    Look for this link on the new repository page

  3. Drag all your files into the upload area

    Include: starter.html, starter.css, and any other files

  4. Rename your files first!

    Before uploading, rename starter.html ‚Üí index.html and starter.css ‚Üí styles.css

  5. Click "Commit changes"

    Your files are now saved online!

3C. Connect to Cloudflare Pages

  1. Go to Cloudflare and create a free account
  2. Click "Workers & Pages" in the sidebar
  3. Click "Create" ‚Üí "Pages" ‚Üí "Connect to Git"
  4. Sign in to GitHub and select your repository
  5. Set these build settings:
    Build commandnpm run build
    Build output directorydist
  6. Click "Save and Deploy"

Your website will be available at:
https://your-project-name.pages.dev

Every time you update files on GitHub, your site updates automatically!

Need Help?

My changes aren't showing up

Make sure you saved the file (Ctrl+S / Cmd+S), then refresh the browser (F5).

The site looks broken

Make sure you renamed starter.css to styles.css ‚Äî the HTML file looks for styles.css!

I can't find my files

Check your Downloads folder. The ZIP file extracts to a folder called web-workshop-starter-main.

Cloudflare build failed

Make sure build command is exactly npm run build and output directory is dist.
